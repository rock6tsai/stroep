Tweener Transition Pack for flashflowfactory

1. I have included an SWC of Tweener inside the package (tweener_1_33_74_as3_swc), which is licensed under MIT. Make sure this is the latest version. http://tweener.googlecode.com 
2. Add this folder to your classpath to enable the transitions. Don't forget to add the SWC to your project too.
3. Have fun with the default transitions

Special note on easing (only applies for Tweener)
If you want to define easing in your pageSetting, you should use the static functions from 'caurina.transitions.Equations' for example Equations.easeOutElastic
