add()method
public function add(pageName:String, classReference:Class, title:String, wildcard:Boolean = false):void

Registers a page reference with optional title to the pageFactory.

Parameters

pageName:String — The pageName reflects the deeplink url, and should start with forward-slash.
 
classReference:Class — Reference to a class, which should extend the Page class
 
title:String — Optional. Title which is showed as browser title
 
wildcard:Boolean (default = false) — Optional. Normally an url represents one page (because the exact pageName links to a classReference). It is possible to add a wildcard to the url, which means you can add optional directories after the pageName (mostly on dynamic pages).
