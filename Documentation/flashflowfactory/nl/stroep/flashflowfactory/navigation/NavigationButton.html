click()method
public function click(path:*, type:String = internal, ... parameters):void

Easy click function, sets path,type and parameters within one function.

Parameters

path:* — Based on the type, this refferes to the path where you should go after the click.
 
type:String (default = internal) — Use ButtonTypes to define what kind of action the action should be handled on click.
 
... parameters — Optional parameters. In case of ButtonTypes.JAVASCRIPT, ButtonTypes.FUNCTION and ButtonTypes.Event; you can pass optional (function) parameter here. In case of ButtonTypes.EXTERNAL you can pass a window-target.
